## 2. 接口清单

| 接口地址 | 方法 | 接口名称 | 功能 |
|---|---|---|---|
| `/api/auth/password-reset/request` | POST | 请求密码重置 | 生成一次性令牌并投递 QQ 邮件 |
| `/api/auth/password-reset/confirm` | POST | 确认密码重置 | 校验令牌并更新密码 |
| `/api/users/me/password` | POST | 修改密码 | 校验旧密码、更新密码并撤销会话 |
| `/api/users/me/sessions` | GET | 查看认证会话 | 返回当前用户设备会话元数据 |
| `/api/users/me/sessions/{id}` | DELETE | 撤销单个会话 | 撤销指定认证会话 |
| `/api/users/me/sessions/revoke-all` | POST | 撤销全部会话 | 撤销当前用户全部认证会话 |
| `/api/users/me/avatar` | POST/DELETE | 头像上传/删除 | 写入或删除 MinIO 私有对象 |
| `/api/users/me/export` | POST | 创建数据导出任务 | 创建异步导出任务 |
| `/api/users/me/deletion` | POST | 提交账号注销 | 二次确认后创建清理任务并撤销会话 |
| `/api/admin/users` | GET | 管理员用户查询 | `admin`、`operator`、`superadmin` 查询脱敏用户摘要 |

## 3. 实现逻辑

密码修改和密码重置都使用现有 PBKDF2 密码格式。重置令牌只保存 SHA-256 哈希，令牌有效期 15 分钟且只能使用一次；成功修改密码后撤销用户全部认证会话。QQ SMTP 配置通过 `.env` 注入，邮件发送不把授权码写入日志或响应。

头像上传限制 PNG/JPEG/WebP 和 2 MB，先写入 MinIO 私有桶，再写入 `user_avatar_assets` 元数据；删除时删除对象并标记资源删除。导出任务生成用户和资料数据 ZIP 写入 MinIO，注销任务由定时处理器软删除账户关联数据。

## 5. 安全与隐私

- 所有个人数据接口要求登录和 CSRF。
- 不返回密码哈希、会话哈希、CSRF 哈希、SMTP 授权码或 MinIO 密钥。
- 注销需要明确确认语，提交后不可恢复。
